# Paperclip AI Setup

```bash
npm install -g paperclipai
npx paperclipai onboard --yes
npx paperclipai run
```

Set `PAPERCLIP_URL=http://127.0.0.1:3100` (or your Paperclip server URL) to use Paperclip agents instead of direct Claude calls.

Agents in `paperclip.config.json`: DataAnalyst, Quant, RiskManager, Sniper.
